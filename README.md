# Fortera Homes

A premium real estate development website for the upcoming landmark development at 82 Genest St. This modern, responsive web application serves as a marketing platform to generate interest and collect registrations.

## 🏢 Overview

Fortera Homes is a real estate development company focused on building premium quality residential properties. The website serves as a "coming soon" landing page for their flagship development, featuring mission-driven development, premium construction standards, and an interest registration system.

## 🚀 Features

- **Responsive Design** - Optimized for desktop, tablet, and mobile
- **Modern Navigation** - Smooth scrolling with mobile-friendly menu
- **Interactive Contact Form** - Client-side validation with success handling
- **Google Maps Integration** - Embedded location map
- **Scroll Animations** - Intersection Observer-based reveal animations
- **Performance Optimized** - Lazy loading and efficient asset management
- **Accessibility Focused** - ARIA labels and semantic HTML
- **SEO Ready** - Meta tags and search-friendly structure

## 🛠️ Tech Stack

- **HTML5** - Semantic markup with accessibility best practices
- **CSS3** - Modern styling with custom properties, Grid, and Flexbox
- **Vanilla JavaScript** - No framework dependencies for optimal performance
- **Google Fonts** - Cormorant Garamond and Montserrat typography

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in a modern web browser
3. No build process or dependencies required
